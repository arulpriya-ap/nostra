# 🛍️ Nostra - Fashion E-commerce Website

## 📌 Project Overview

**Nostra** is a responsive fashion e-commerce website developed using **HTML5, CSS3 (Tailwind CSS), and JavaScript**. The project provides users with a modern online shopping experience by showcasing fashion collections, new arrivals, and most wanted products.

The website is fully responsive and includes interactive features such as an image carousel, product search, product filtering, navigation between pages, and a wishlist functionality. This project was built to strengthen front-end development skills by implementing real-world e-commerce functionalities using JavaScript.

---

## 🚀 Features

### 🏠 Home Page

* Responsive navigation bar
* Promotional announcement banner with close functionality
* Hero image slider (carousel)
* New Arrivals section
* Most Wanted products section
* Limited Offer section
* Newsletter subscription section
* Footer with social media icons

### 🛍️ Collections Page

* Responsive product grid layout
* Search products by:

  * Product Name
  * Product Price
* Filter products by:

  * Occasion
  * Color
  * Arrival Type
* Responsive sidebar filter section

### ❤️ Wishlist Feature

* Click the white heart icon to add a product to the wishlist.
* Heart icon changes from white to red.
* Clicking the red heart removes the product from the wishlist.

### 🎠 Image Carousel

* Previous and Next navigation buttons
* Smooth sliding transition
* Responsive image display
* Infinite image navigation

### 🔍 Product Search

* Real-time search functionality using JavaScript
* Search products by:

  * Product Name
  * Product Price
* Case-insensitive search

### 📱 Responsive Design

* Mobile-first responsive layout
* Built using Tailwind CSS
* Optimized for:

  * Mobile
  * Tablet
  * Desktop
